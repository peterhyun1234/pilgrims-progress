## 1. MVP 범위 정의 (Scope Definition)

### 1.1 개요

MVP는 무료 데모로 배포되며, 원작의 가장 강력한 서사 아크인 **"죄의 자각 → 방황 → 은혜 → 구원"**을 완결된 형태로 전달한다. 플레이어는 크리스천이 되어 멸망의 도시에서 출발하여 십자가 앞에서 짐을 내려놓는 순간까지의 여정을 체험한다.

| 항목 | 내용 |
|------|------|
| **빌드 명칭** | 천로역정: 순례자의 여정 — 데모 (Demo) |
| **서사 범위** | 원작 섹션 {10}~{94} |
| **예상 플레이타임** | 40~60분 (1회 플레이 기준) |
| **리플레이 가치** | 2~3회 (분기에 따라 다른 대화/장면 체험) |
| **플랫폼** | PC (Windows 10/11, macOS 12+) |
| **언어** | 한국어 단일 |
| **가격** | 무료 |

### 2.1 멸망의 도시 (City of Destruction)

#### 장면 설명

| 항목 | 내용 |
|------|------|
| **분위기** | 무거운 회색빛 하늘, 불안한 정적. 도시는 겉으로는 평범하지만 내면적으로 위태로움 |
| **컬러 톤** | 탁한 황토색, 회갈색, 먹구름의 짙은 남색. 어둠의 팔레트 기반 |
| **시간대** | 해 질 녘 → 밤 → 새벽 (장면 내 시간 흐름) |
| **주요 비주얼** | 누더기 옷의 크리스천, 등에 진 거대한 짐, 손에 든 책(성경), 탁한 하늘 아래 초라한 집 |
| **배경 음악** | 낮고 불안한 현악 + 먼 천둥 소리 → 전도자 등장 시 희미한 빛의 모티프 |

#### 등장 캐릭터 & 초상화

| 캐릭터 | 필요 표정 | 설명 |
|--------|----------|------|
| 크리스천 | 슬픔, 두려움, 결의 | 초반 울고 있는 모습 → 전도자의 말에 결의를 다지는 표정 전환 |
| 크리스천의 아내 | 걱정, 냉담 | 처음에는 걱정하다가 점점 냉담해짐 |
| 전도자 | 진지, 격려 | 양피지 두루마리를 건네며 길을 가리키는 모습 |

#### 대화 흐름 요약

```
[오프닝 시네마틱]
  크리스천이 책을 읽으며 울고 떨림
  내레이터: "이 세상이라는 광야를 걸어가던 중..."
  크리스천: "내가 어찌하여야 하리이까?"

[집 안 장면]
  크리스천 → 아내와 자녀에게 고백
  "오 나의 사랑하는 아내여... 우리 도시가 불로 소멸될 것이오"
  아내 반응: 조롱/걱정/무시 (고정)

[들판 장면 — 전도자 등장]
  크리스천: "구원을 받으려면 내가 어찌하여야 하리이까?"
  전도자: "저기 빛나는 빛이 보이느냐?"
  전도자가 양피지 두루마리를 건넴

[선택지 1 — 첫 번째 결단]
  → 플레이어 선택
```

### 2.2 들판 (The Field) — 완고와 유연

#### 장면 설명

| 항목 | 내용 |
|------|------|
| **분위기** | 광활한 벌판. 크리스천이 달리고, 뒤에서 두 사람이 쫓아옴 |
| **컬러 톤** | 새벽빛의 회청색 → 점차 밝아지는 자연색 |
| **주요 비주얼** | 넓은 들판, 멀리 희미한 빛(좁은 문 방향), 달려오는 완고와 유연 |
| **배경 음악** | 긴장감 있는 빠른 템포의 곡 → 대화 시 차분해짐 |

#### 등장 캐릭터 & 초상화

| 캐릭터 | 필요 표정 | 설명 |
|--------|----------|------|
| 크리스천 | 결의, 열정 | 달리면서도 천국의 영광을 설명할 때 빛나는 표정 |
| 완고 | 분노, 경멸 | "쯧! 당신의 책 따위는 치워라" |
| 유연 | 호기심, 기대, 불안 | 크리스천의 말에 감동받는 모습 → 후에 불안 |

#### 대화 흐름 요약

```
[추격 장면]
  완고 & 유연이 크리스천을 따라잡음
  "당신을 설득하여 우리와 함께 돌아가려고"

[설득 대화]
  크리스천이 천국의 보물을 설명
  완고는 거부, 유연은 마음이 움직임

[선택지 — 동행자 설득]
  → 플레이어 선택

[결과]
  완고: 분노하며 돌아감
  유연: 크리스천과 동행 시작
  천국의 영광에 대한 대화
```

### 2.3 절망의 늪 (Slough of Despond)

#### 장면 설명

| 항목 | 내용 |
|------|------|
| **분위기** | 갑작스러운 공포. 발밑이 무너지며 질퍽한 늪에 빠지는 절망적 상황 |
| **컬러 톤** | 탁한 갈색, 진흙색, 어두운 올리브. 안개가 자욱 |
| **주요 비주얼** | 질퍽한 늪, 빠져나가려 허우적대는 크리스천, 등의 짐이 그를 가라앉힘 |
| **배경 음악** | 낮고 무거운 베이스 드론 + 불규칙한 심장 박동 같은 리듬 |
| **게임플레이** | **QTE (Quick-Time Event)** — 타이밍에 맞춰 버튼을 눌러 허우적거림 표현 |

#### 등장 캐릭터 & 초상화

| 캐릭터 | 필요 표정 | 설명 |
|--------|----------|------|
| 크리스천 | 공포, 절망, 안도 | 늪에 빠져 공포 → 도움에 의해 구출되며 안도 |
| 유연 | 분노, 배신감 | "이것이 당신이 말한 행복이란 것이오?" |
| 도움 | 침착, 자비 | "네 손을 내밀어라" |

#### 대화 흐름 요약

```
[추락 시퀀스]

[유연의 이탈]
  유연: "이것이 당신이 말한 행복이란 것이오?"
  유연이 분노하며 자기 집 쪽으로 빠져나감

[QTE 계속 — 크리스천 홀로]
  짐 때문에 가라앉지만 좁은 문 방향을 향해 애씀

[도움 등장]
  도움: "거기서 무엇을 하고 있느냐?"
  도움이 손을 잡아 끌어올림

[선택지 — 도움과의 대화]
  → 플레이어 선택
```

### 3.2 주요 분기 경로

MVP에는 하나의 **대분기(Major Branch)**가 존재한다:

```
                    ┌─ [A] 유혹 수락 ──→ 시내산 경로 ──→ 전도자 질책 ──→ 좁은 문
C04-01 ─────────────┤
(세상지혜씨)        ├─ [B] 거부 ──────→ (시내산 건너뜀) ──────────────→ 좁은 문
                    │
                    └─ [C] 질문 (숨김) ─→ 논리 파헤침 ──→ 자동 거부 ──→ 좁은 문
```

**경로 A (유혹 수락)**: 원작의 전체 시내산 시퀀스를 체험. 전도자의 질책과 회개를 통해 깊은 교훈. 플레이타임 +4~6분.

**경로 B (거부)**: 시내산을 건너뛰고 바로 좁은 문으로 향함. 짧은 내레이션: "크리스천은 세상지혜씨의 교묘한 말을 뿌리치고 빛을 향해 계속 달렸다." 보너스 성경 구절 획득.

**경로 C (질문 — 숨김)**: 세상지혜씨의 논리를 지혜로 파헤치는 특별 대화. 자동으로 거부로 전환되며 높은 지혜 보너스.

> 모든 경로는 좁은 문에서 합류하며, 이후는 동일하게 진행된다. 선의와의 대화에서 C04-01A 경로를 선택한 경우, 세상지혜씨를 따랐던 고백이 추가된다.

### 3.3 숨김 선택지 해금 조건 요약

| 숨김 선택지 | 요구 스탯 | 효과 |
|------------|----------|------|
| C01-02C "잠시 더 생각해본다" | 없음 (초기 접근 가능이나 C 옵션) | 전도자의 추가 격려 |
| C02-02C 잠언 인용 | 지혜 ≥ 20 | 지혜 +10, 추가 대화 |
| C03-02C 계단에 대한 질문 | 지혜 ≥ 25 | 지혜 +10, 추가 구절 |
| C04-01C 세상지혜씨에게 질문 | 지혜 ≥ 30 | 지혜 +10, 시내산 자동 건너뜀 |
| C05-02C "은혜를 구합니다" | 믿음 ≥ 30 | 믿음 +15, 요 6:37 구절 |
| C06-R5-01C "함께 싸우겠습니다!" | 용기 ≥ 40 | 용기 +15, 특별 연출 |
| C07-01C 말없이 무릎 꿇고 울음 | 믿음 ≥ 50 | 믿음 +20, 가장 깊은 감동 연출 |

## 6. 수용 기준 (Acceptance Criteria)

MVP가 완료되었다고 판단하기 위한 테스트 가능한 기준 목록이다.

### 6.1 핵심 플레이 경험

- [ ] 플레이어가 멸망의 도시에서 시작하여 십자가까지의 전체 여정을 크래시 없이 완주할 수 있다
- [ ] 전체 플레이타임이 40~60분 범위 내에 있다 (1회 플레이 기준)
- [ ] 세상지혜씨 분기의 양쪽 경로(수락/거부) 모두 완주 가능하다
- [ ] 모든 7개 장소가 올바른 순서로 진행된다
- [ ] 해석자의 집 7개 방이 모두 방문 가능하다

### 6.2 선택지 & 분기

- [ ] 26개 선택 포인트가 모두 정상 작동한다
- [ ] 모든 선택지의 텍스트가 올바르게 표시된다
- [ ] 7개 숨김 선택지가 올바른 스탯 조건에서만 나타난다
- [ ] C04-01 대분기(시내산 수락/거부)가 올바르게 분기되고 합류된다
- [ ] 경로 A(시내산)에서 전도자 질책 시퀀스가 정상 재생된다
- [ ] 경로 B(거부)에서 시내산이 건너뛰어지고 대체 내레이션이 표시된다
- [ ] 선택 결과에 따른 후속 대화 변화가 올바르게 적용된다

### 6.3 스탯 시스템

- [ ] 믿음/용기/지혜 3개 스탯이 초기값 10으로 시작한다
- [ ] 모든 선택지에서 올바른 스탯 변화가 적용된다 (+/− 값)
- [ ] 스탯 변화 시 UI에 알림이 표시된다
- [ ] 스탯 UI 패널에서 현재 값을 확인할 수 있다
- [ ] 스탯이 0 미만으로 떨어지지 않는다
- [ ] 숨김 선택지가 정확한 스탯 임계값에서 해금된다

### 6.4 수집 시스템

- [ ] 21개 성경 구절 카드가 모두 올바른 조건에서 수집 가능하다
- [ ] 수집된 구절이 도감에 올바르게 기록된다
- [ ] 구절 수집 시 알림 팝업이 표시된다
- [ ] 14명의 캐릭터가 만남 시 도감에 자동 등록된다
- [ ] 캐릭터 도감에 이름, 상징, 핵심 대사가 올바르게 표시된다
- [ ] 여정 일지에 방문한 장소와 주요 선택이 기록된다
- [ ] 도감/일지 UI에서 수집 현황을 한눈에 볼 수 있다

### 6.5 QTE (절망의 늪)

- [ ] QTE 프롬프트가 화면에 올바르게 표시된다
- [ ] 타이밍 입력이 올바르게 인식된다 (키보드/마우스)
- [ ] 성공/부분 실패에 따른 연출 차이가 정상 작동한다
- [ ] QTE 실패해도 게임이 진행 가능하다 (데드엔드 없음)
- [ ] QTE 지속 시간이 20~30초 범위 내이다

### 6.6 저장 & 불러오기

- [ ] 수동 저장 3슬롯이 정상 작동한다
- [ ] 자동 저장이 장소 전환 시 트리거된다
- [ ] 자동 저장이 주요 선택 직후 트리거된다
- [ ] 저장 파일에서 불러오기 시 정확한 위치에서 재개된다
- [ ] 불러오기 시 스탯, 수집 현황, 선택 이력이 복원된다
- [ ] 다른 슬롯 간 저장/불러오기가 간섭하지 않는다
- [ ] 저장 파일 손상 시 적절한 오류 메시지가 표시된다

### 6.7 UI & UX

- [ ] 메인 메뉴에서 새 게임/이어하기/설정/종료가 작동한다
- [ ] 일시정지 메뉴에서 저장/불러오기/설정/메인메뉴 복귀가 가능하다
- [ ] 텍스트 속도 설정이 정상 적용된다 (느림/보통/빠름/즉시)
- [ ] 자동 진행 모드가 정상 작동한다
- [ ] 대화 히스토리 로그에서 이전 대화를 스크롤하여 볼 수 있다
- [ ] 내레이터 텍스트와 캐릭터 대화가 시각적으로 구분된다
- [ ] 모든 UI 요소가 1280×720에서 3840×2160까지 올바르게 스케일링된다
- [ ] 전체화면/창모드/보더리스 전환이 정상 작동한다

### 6.8 오디오

- [ ] 모든 장소에서 올바른 BGM이 재생된다
- [ ] 장소 전환 시 BGM이 자연스럽게 페이드/전환된다
- [ ] 모든 SFX가 올바른 타이밍에 재생된다
- [ ] 마스터/BGM/SFX 볼륨이 독립적으로 조절 가능하다
- [ ] 볼륨 설정이 저장/복원된다
- [ ] 음소거 기능이 정상 작동한다

### 6.9 시각 & 연출

- [ ] 모든 27개 배경이 올바르게 표시된다
- [ ] 모든 33개 캐릭터 표정이 올바르게 전환된다
- [ ] 수채화 번짐 전환 효과가 장소 이동 시 재생된다
- [ ] 십자가 시네마틱 시퀀스가 완전히 재생된다 (짐 떨어짐, 빛의 입자, 새 옷)
- [ ] 시내산 불꽃/화면 흔들림 효과가 정상 작동한다
- [ ] 심판 비전 시퀀스가 정상 재생된다
- [ ] 파티클 효과가 성능에 악영향을 주지 않는다

### 6.10 성능 & 안정성

- [ ] 최소 사양 PC에서 30fps 이상 유지된다
- [ ] 권장 사양 PC에서 60fps 이상 유지된다
- [ ] 메모리 누수가 없다 (전체 플레이 중 메모리 사용량 안정적)
- [ ] Windows 빌드가 크래시 없이 전체 플레이 가능하다
- [ ] macOS 빌드가 크래시 없이 전체 플레이 가능하다
- [ ] 빌드 크기가 2GB 이하이다

### 6.11 데모 종료 & 전환

- [ ] 십자가 시퀀스 완료 후 데모 종료 화면이 표시된다
- [ ] 데모 종료 화면에 정식판 안내 정보가 포함된다
- [ ] 데모 종료 후 메인 메뉴로 돌아갈 수 있다
- [ ] 데모 종료 후 처음부터 다시 플레이 가능하다

## 7. 알려진 제한사항 (Known Limitations)

MVP 데모는 다음 사항을 **의도적으로 포함하지 않는다**:

### 7.1 플랫폼

| 제한사항 | 설명 | 정식판 계획 |
|----------|------|------------|
| 모바일 미지원 | iOS/Android 빌드 없음 | 정식판 이후 모바일 포팅 |
| 콘솔 미지원 | Nintendo Switch, PlayStation, Xbox 없음 | 미정 |
| 웹 미지원 | WebGL 빌드 없음 | 검토 중 |

### 7.2 언어 & 접근성

| 제한사항 | 설명 | 정식판 계획 |
|----------|------|------------|
| 한국어 단일 | 영어 등 다른 언어 미지원 | 정식판에서 한영 동시 지원 |
| 성우 녹음 없음 | 텍스트만으로 대화 진행 | 예산 확보 시 핵심 대사만 녹음 검토 |
| 제한된 접근성 | 색맹 모드, 스크린리더 미지원 | 정식판에서 기본 접근성 옵션 추가 |
| 자막 크기 조절 불가 | 고정 텍스트 크기 | 정식판에서 텍스트 크기 설정 추가 |

### 7.3 콘텐츠

| 제한사항 | 설명 | 정식판 계획 |
|----------|------|------------|
| Chapter 01~02만 포함 | 원작 전체의 약 10% | 정식판에서 Part 1 전체 (25개 장소) |
| 싱글 엔딩 | 짐이 떨어지는 단일 결말 (변형 연출만 존재) | 정식판에서 멀티플 엔딩 |
| 전투 시스템 미포함 | 아폴리온 전투 등 없음 (QTE만) | 정식판에서 QTE 기반 전투 시스템 |
| 캐릭터 관계 시스템 없음 | NPC와의 관계도 미구현 | 정식판에서 동행 시스템 |

### 7.4 온라인 & 소셜

| 제한사항 | 설명 | 정식판 계획 |
|----------|------|------------|
| 온라인 기능 없음 | 클라우드 저장, 리더보드 없음 | 검토 중 |
| 공유 기능 없음 | 스크린샷/클립 공유 기능 없음 | 정식판에서 추가 검토 |
| 업적 시스템 없음 | Steam 업적 미연동 | 정식판 Steam 출시 시 추가 |
| 분석/텔레메트리 없음 | 플레이 데이터 수집 없음 | 검토 중 (GDPR 준수) |

| 제한사항 | 설명 | 정식판 계획 |
