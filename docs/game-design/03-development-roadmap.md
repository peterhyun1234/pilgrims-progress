### Phase 1: 수직 절단면 (Vertical Slice) — 4~6주

**목표**: "해석자의 집" 하나를 완벽하게 만들어 전체 파이프라인을 검증한다.

**왜 해석자의 집인가?**
- 7개의 방(Rooms)으로 구성 → 내부 이동 시스템 테스트에 적합
- 대화, 선택지, 스탯 변화, 상징 해석 등 핵심 메카닉이 모두 포함
- 다양한 분위기의 아트 에셋 필요 → 아트 파이프라인 검증
- 중간 난이도의 연출 → 기준점으로 적절

| 시스템 | 구현 내용 | 검증 항목 |
|--------|----------|----------|
| **대화 시스템** | Ink → Unity 텍스트 표시, 선택지 UI, 캐릭터 초상화 표시 | 한글 렌더링, 타이핑 효과, 선택지 UI |
| **선택 & 분기** | 7개 방에서 최소 10개 선택지, 2개 이상 분기 | 분기 후 합류, 플래그 저장 |
| **스탯 시스템** | 믿음/용기/지혜 표시, 선택에 따른 변동 | UI 갱신, 스탯 기반 조건 분기 |
| **장면 전환** | 방 간 이동 시 페이드/디졸브 전환 | 전환 부드러움, 로딩 없음 |
| **아트 에셋** | 해석자의 집 배경 7장, 해석자 캐릭터 초상화 3장(표정), UI 프레임 | AI 생성 → 정제 워크플로 시간 측정 |
| **오디오** | BGM 1곡 (잔잔한 실내 테마), SE 3~5개 (문 열림, 선택, 전환) | 볼륨 밸런스, 루프 자연스러움 |
| **저장/불러오기** | 기본 세이브 (Ink 상태 + 게임 상태 직렬화) | 저장 후 복원 시 상태 일치 확인 |

**Phase 1 종료 기준 (Exit Criteria)**:
- [ ] 해석자의 집 7개 방 모두 플레이 가능
- [ ] 처음부터 끝까지 막힘 없이 15~20분 플레이 완료
- [ ] 외부인 1~2명 테스트 플레이 피드백 수집
- [ ] 에셋 1세트(배경 7장 + 캐릭터 3장) 제작 소요 시간 측정 완료

> **핵심 질문**: "이 경험이 재미있는가? 감동을 주는가?" → **Go/No-Go 결정점 ①**

### 4.2 Go/No-Go 결정 기준

#### ★ 결정점 ①: 수직 절단면 후 (W12)

| 평가 항목 | Go 조건 | No-Go 시 대응 |

| 기술 안정성 | 크래시/버그 없이 15분 플레이 가능 | 기술 부채 해결 후 재시도 |
| 아트 품질 | AI 정제 결과가 스타일 가이드와 일치 | 프롬프트/정제 파이프라인 수정 |
| 생산 속도 | 배경 1장 평균 2~4시간 이내 | 에셋 수 축소 또는 스타일 간소화 |
| 본인 동기 | 계속 만들고 싶은 열정이 있는가 | 프로젝트 보류/축소 |

| 평가 항목 | Go 조건 | No-Go 시 대응 |
