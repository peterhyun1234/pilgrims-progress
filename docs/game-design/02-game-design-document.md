## 1. 핵심 게임 루프 (Core Game Loop)

### 1.1 플레이어가 매 순간 하는 일

플레이어는 크리스천이 되어 멸망의 도시에서 천성까지의 순례길을 걸어간다. 모든 장소에서 반복되는 핵심 행동 사이클은 다음과 같다:

```
┌─────────────────────────────────────────────────────────────────┐
│                      핵심 게임 루프 다이어그램                      │
│                                                                 │
│   ┌──────────┐    ┌──────────────┐    ┌───────────────┐         │
│   │ 1. 장소  │───▶│ 2. 발견     │───▶│ 3. 대화 &    │         │
│   │  탐험    │    │ (캐릭터/    │    │  선택        │         │
│   │          │    │  아이템)    │    │              │         │
│   └──────────┘    └──────────────┘    └──────┬────────┘         │
│        ▲                                     │                  │
│        │                                     ▼                  │
│   ┌──────────┐                        ┌───────────────┐         │
│   │ 5. 다음  │◀───────────────────────│ 4. 스탯 변화  │         │
│   │  장소로  │                        │ & 결과        │         │
│   │  이동    │                        │              │         │
│   └──────────┘                        └───────────────┘         │
│                                                                 │
│   * 특수 장소에서는 3과 4 사이에 [전투/챌린지]가 삽입됨           │
│   * 수집 요소는 2에서 발생 (성경 구절 카드, 숨겨진 대화)          │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 루프 단계별 상세

#### 1단계: 장소 탐험 (Explore Location)

플레이어가 새 장소에 도착하면 다음이 발생한다:

| 요소 | 설명 |
|------|------|
| **배경 전환** | 수채화 번짐 효과(Dissolve)로 새 장소 배경이 펼쳐짐 |
| **장소명 표시** | 화면 중앙에 한국어 장소명 + 영문 원제가 페이드인 (예: "절망의 늪 / Slough of Despond") |
| **내레이션** | 꿈꾸는 자(화자)의 시점으로 짧은 묘사 텍스트 출력 |
| **BGM 전환** | 장소 분위기에 맞는 배경 음악으로 크로스페이드 |
| **탐험 포인트** | 배경 내 인터랙티브 요소에 빛나는 점 표시 (탭/클릭 가능) |

탐험 포인트의 종류:

| 포인트 유형 | 아이콘 | 상호작용 결과 |
|------------|--------|-------------|
| **인물** | 사람 실루엣 | 대화 개시 |
| **물건** | 빛나는 원 | 아이템/성경 구절 카드 획득 |
| **경치** | 눈 모양 | 내레이션 텍스트 (세계관 보충) |
| **길** | 화살표 | 다음 장소로 이동 (조건 충족 시) |

#### 2단계: 발견 (Discover Characters/Items)

탐험 포인트를 클릭하면 캐릭터 만남, 아이템 발견, 또는 추가 서사가 전개된다. 주요 캐릭터는 반드시 만나야 하며(메인 퀘스트), 성경 구절 카드와 숨겨진 대화는 선택적이다(사이드 컬렉션).

#### 3단계: 대화 & 선택 (Dialogue & Choices)

대화 시스템은 Ink 엔진으로 구동된다. 플레이어는 크리스천의 대사를 선택한다. 선택지 유형은 다음과 같다:

| 선택지 유형 | 설명 | 예시 |
|------------|------|------|
| **도덕적 딜레마** | 옳고 그름이 명확하지 않은 선택 | 세상지혜씨의 충고를 따를 것인가? |
| **탐험적 선택** | 추가 정보 획득 여부 | 해석자의 집에서 더 보여달라고 요청할 것인가? |
| **지식 확인** | 이전에 배운 내용 기반 | 형식과 위선에게 좁은 문을 언급할 것인가? |
| **감정 표현** | 캐릭터의 감정 방향 결정 | 십자가 앞에서 울 것인가, 기쁨으로 뛸 것인가? |
| **스탯 게이팅** | 일정 스탯 이상이어야 등장 | 용기 40 이상 → "사자를 향해 담대히 걸어간다" 옵션 |

#### 4단계: 스탯 변화 & 결과 (Stat Changes & Consequences)

선택에 따라 믿음(Faith), 용기(Courage), 지혜(Wisdom) 스탯이 변화한다. 변화량은 선택의 중요도에 비례한다.

| 변화 규모 | 범위 | 발생 빈도 | 예시 |
|-----------|------|----------|------|
| **소** | ±3~5 | 매 대화 | 경건과의 대화에서 솔직하게 답변 → 지혜 +3 |
| **중** | ±8~12 | 주요 갈림길 | 시내산으로 가는 대신 좁은 문으로 되돌아감 → 믿음 +10 |
| **대** | ±15~20 | 챕터 핵심 이벤트 | 아폴리온과의 전투 승리 → 용기 +20, 믿음 +15 |

스탯 변화 시 HUD에 짧은 알림이 표시된다: `[믿음 +10 ▲]`

#### 5단계: 다음 장소로 이동 (Move to Next Location)

현재 장소의 메인 퀘스트를 완료하면 다음 장소로의 이동이 해금된다. 이동 시:

- 맵 화면에서 경로 애니메이션 재생 (점선이 실선으로 변함)
- 이동 중 짧은 내레이션 또는 크리스천의 독백
- 특정 구간에서는 이동 중 랜덤 이벤트 발생 가능 (예: 무지가 뒤에서 따라옴)

### 1.3 세션 구조

하나의 "챕터"는 2~4개 장소로 구성되며, 약 20~40분의 플레이타임을 갖는다.

| 챕터 | 장소 | 예상 플레이타임 |
|------|------|---------------|
| Ch.1 | 멸망의 도시 → 절망의 늪 → 세상지혜씨 → 좁은 문 | 20~30분 |
| Ch.2 | 해석자의 집 → 십자가/무덤 | 15~25분 |
| Ch.3 | 어려움의 언덕 → 아름다운 궁전 | 20~30분 |
| Ch.4 | 겸손의 골짜기 → 사망의 음침한 골짜기 | 25~35분 |
| Ch.5 | 신실과의 동행 → 수다쟁이 대화 | 15~20분 |
| Ch.6 | 허영의 시장 → 소망과의 동행 | 25~35분 |
| Ch.7 | 의심의 성 → 기쁨의 산 | 25~35분 |
| Ch.8 | 마법의 땅 → 천성 | 30~40분 |

**총 예상 플레이타임**: 3~5시간 (Part 1 전체)

## 2. 서사 시스템 상세 (Narrative System)

### 2.1 Ink 통합 아키텍처

모든 게임 서사는 Inkle의 Ink 스크립팅 언어로 작성되며, Unity의 Ink Integration 플러그인을 통해 런타임에 구동된다.

**파일 구조**:

```
Assets/
  InkStories/
    globals.ink          ← 전역 변수 (스탯, 플래그, 수집품)
    ch01_destruction.ink ← 챕터 1: 멸망의 도시 ~ 좁은 문
    ch02_interpreter.ink ← 챕터 2: 해석자의 집 ~ 십자가
    ch03_difficulty.ink  ← 챕터 3: 어려움의 언덕 ~ 궁전
    ch04_valleys.ink     ← 챕터 4: 겸손/사망의 골짜기
    ch05_faithful.ink    ← 챕터 5: 신실, 수다쟁이
    ch06_vanity.ink      ← 챕터 6: 허영의 시장
    ch07_doubting.ink    ← 챕터 7: 의심의 성 ~ 기쁨의 산
    ch08_celestial.ink   ← 챕터 8: 마법의 땅 ~ 천성
```

### 2.3 대화 분기 구조

Ink에서 대화는 `knot` (큰 장면)과 `stitch` (세부 흐름)로 구성된다. 선택지는 `*` (일회성) 또는 `+` (반복 가능)으로 표기한다.

**예시: 멸망의 도시에서의 출발 장면**

```ink
=== city_of_destruction ===

# LOCATION: 멸망의 도시
# BGM: ambient_city_dusk

꿈속에서 나는 누더기 옷을 입은 한 사람을 보았다.
그는 자기 집을 등지고 서 있었으며, 손에는 책 한 권을 들고 있었고, 등에는 커다란 짐을 지고 있었다.

그가 책을 펴서 읽자, 읽으면서 울었고 떨었다.

# SPEAKER: 크리스천
# EMOTION: despair

"내가 어찌하여야 하리이까?"

-> family_confrontation

= family_confrontation
# SPEAKER: narrator

크리스천은 집으로 돌아가 아내와 자녀들에게 마음을 열었다.

# SPEAKER: 크리스천
# EMOTION: urgent

"오 나의 사랑하는 아내여, 내 뱃속에서 난 자녀들아—"
"이 도시가 하늘에서 내리는 불로 소멸될 것이오."

# SPEAKER: narrator
가족들은 놀랐으나, 그의 말을 믿지 않았다. 그의 머리에 광기가 들었다고 여겼다.

* [가족을 설득하려 더 노력한다] -> persuade_family
    ~ wisdom += 5
    ~ journal_entries += 1
* [혼자 방으로 물러나 기도한다] -> pray_alone
    ~ faith += 5
    ~ journal_entries += 1
* [즉시 들판으로 뛰쳐나간다] -> flee_immediately
    ~ courage += 3

= persuade_family
# SPEAKER: 크리스천
# EMOTION: pleading

"제발, 나의 말을 들어 주오. 이 책에 기록된 것은 사실이오—"

# SPEAKER: 아내
그의 아내가 차갑게 말했다.
"또 그 이상한 소리... 어서 자거라. 아침이면 나을 것이다."

{wisdom >= 30:
    # SPEAKER: narrator
    크리스천은 지혜롭게 물러났다. 지금은 때가 아님을 알았다.
    -> pray_alone
- else:
    # SPEAKER: narrator
    크리스천은 거듭 말했으나 가족은 마음이 더 완고해져 갔다.
    -> pray_alone
}

= pray_alone
# SPEAKER: narrator
크리스천은 자기 방으로 물러나 기도했다.
그리고 홀로 들에 나가 때로는 읽고, 때로는 기도하며 날들을 보냈다.

~ faith += 3
~ burden_awareness += 10

# BIBLE_CARD: 행 2:37
# CARD_TEXT: "그들이 이 말을 듣고 마음에 찔려 베드로와 나머지 사도에게 물어 이르되 형제들아 우리가 어찌할꼬 하거늘"

-> meet_evangelist
```

### 2.4 스탯 게이팅 예시

특정 스탯이 임계값 이상일 때만 나타나는 선택지:

```ink
=== wicket_gate ===

# SPEAKER: 선의
"네 짐에 대해서는 구원의 장소에 이를 때까지 참고 지고 가라."

* [감사히 받아들인다] -> accept_burden
    ~ faith += 8
* {courage >= 40} [짐을 지금 벗을 수 없는지 다시 한번 여쭙니다] -> ask_about_burden
    ~ wisdom += 5
* {wisdom >= 50} [그 "구원의 장소"가 정확히 어디인지 묻는다] -> ask_about_cross
    ~ wisdom += 8
    ~ faith += 3

= ask_about_cross
# SPEAKER: 선의
# EMOTION: warm

선의가 따뜻하게 미소 지으며 말했다.
"아직 말할 때가 아니다. 그러나 마음을 가라앉혀라—"
"때가 되면 반드시 알게 될 것이다. 그리고 그 기쁨은..."
"지금 기대하는 것보다 훨씬 클 것이니라."

~ faith += 5
-> accept_burden
```

### 2.5 분기 구조 패턴

본 게임은 크게 세 가지 분기 패턴을 사용한다:

| 패턴 | 구조 | 사용 위치 |
|------|------|----------|
| **수렴형 (Converging)** | 선택지가 다르지만 결국 같은 결과로 합류. 스탯만 다르게 변화 | 대부분의 대화 선택 |
| **갈래형 (Branching)** | 선택에 따라 다른 장면을 경험. 최종적으로는 합류 | 세상지혜씨 → 시내산 우회 vs 직행 |
| **게이팅형 (Gating)** | 스탯이 부족하면 특정 선택지가 잠김. 회색 처리로 존재 자체는 보여줌 | 궁전에서의 심층 대화, 기쁨의 산 조망 |

**분기 다이어그램 (Ch.1~2 MVP)**:

```
멸망의 도시
    │
    ├── [가족 설득] ──┐
    ├── [기도]  ──────┤
    └── [즉시 출발] ──┘
           │
       전도자 만남
           │
    ┌──────┴──────┐
    │             │
 완고 & 유연    (직행)
    │             │
    └──────┬──────┘
           │
      절망의 늪
    ┌──────┴──────┐
    │             │
  [도움 요청]  [홀로 탈출 시도]
    │             │
    └──────┬──────┘
           │
    ┌──────┴──────────┐
    │                 │
  [세상지혜씨      [직행]
   따라감 ─→         │
   시내산 ─→         │
   전도자 질책       │
   ─→ 돌아옴]       │
    │                 │
    └──────┬──────────┘
           │
        좁은 문
           │
      해석자의 집
    (7개 방 순차 탐험)
           │
       ★ 십자가 ★
     (짐이 떨어짐)
```

### 2.6 Ink 태그 시스템

Unity와 Ink 간 통신을 위해 다음 태그 규칙을 사용한다:

| 태그 | 용도 | 예시 |
|------|------|------|
| `# LOCATION:` | 장소 전환 | `# LOCATION: 절망의 늪` |
| `# BGM:` | 배경 음악 전환 | `# BGM: slough_ambient` |

### 3.2 짐 (Burden) 메카닉

짐은 크리스천의 등에 진 죄의 무게를 나타내는 특수 시스템이다. 게임 시작부터 십자가까지 지속되며, 게임플레이에 실질적 영향을 준다.

#### 짐의 게임플레이 효과

| 짐 무게 | 상태 | 이동 속도 | 대화 제한 | 시각 효과 |
|---------|------|----------|----------|----------|
| **100** (시작) | 극중 | 매우 느림 | 절망적 대사 위주 | 화면 약간 어둡게, 어깨 처진 스프라이트 |
| **80~99** | 무거움 | 느림 | 짐에 대한 언급 빈번 | 걸을 때 무거운 발소리 SFX |
| **50~79** | 보통 | 보통 | 일반 대화 가능 | 기본 상태 |
| **30~49** | 가벼워짐 | 빠름 | 희망적 대사 가능 | 화면 약간 밝아짐 |
| **0** (십자가) | **해방** | 자유로움 | 제한 해제, 노래 가능 | 빛 이펙트, 밝은 화면, 새 스프라이트 |

#### 짐 무게 변화 이벤트

| 이벤트 | 짐 변화 | 설명 |
|--------|---------|------|
| 게임 시작 | 100 (고정) | "크리스천의 등에는 커다란 짐이 있었다" |
| 전도자를 만남 | -5 | 탈출의 방향을 아는 것만으로 약간 가벼워짐 |
| 절망의 늪에서 더 깊이 빠짐 | +10 (일시적) | 짐 때문에 더 가라앉는 연출 |
| 세상지혜씨 따라감 | +15 | 잘못된 길로 가면 짐이 더 무거워짐 |
| 전도자 질책 후 돌아옴 | -10 | 회개와 올바른 방향 회복 |
| 좁은 문 통과 | -5 | 아직 짐은 벗겨지지 않음 |
| 해석자의 집 각 교훈 | -3 (×7) | 영적 이해가 깊어질수록 조금씩 가벼워짐 |
| **십자가 도착** | **= 0** | **짐이 풀려 떨어져 무덤으로 굴러감** |

#### 십자가 장면 특수 연출

짐 = 0이 되는 순간은 게임 전체의 정점이다:

1. 화면이 서서히 밝아지며 십자가가 보인다
2. 크리스천이 십자가 가까이 갈수록 짐에서 삐걱거리는 소리가 남
3. 십자가 앞에 서는 순간 — **짐이 풀리는 SFX** (끈이 끊어지는 소리)
4. 짐이 등에서 떨어져 굴러감 — **물리 시뮬레이션 연출**
5. 무덤 입구로 빠져 사라짐 — **화면 완전히 밝아짐**
6. CG 이벤트: 기쁨으로 세 번 뛰어오르는 크리스천
7. 세 빛나는 자의 세 가지 선물 (죄 사함 / 새 옷 / 봉인된 두루마리)
8. 크리스천의 노래: "복된 십자가! 복된 무덤!"

### 3.3 동행자 시스템

십자가 이후, 크리스천은 순차적으로 동행자를 얻는다.

| 동행자 | 합류 시점 | 이탈 시점 | 관계 스탯 | 특수 효과 |
|--------|----------|----------|----------|----------|
| **신실** (Faithful) | Ch.5 시작 | Ch.6 허영의 시장 (순교) | faithful_trust | 용기 보너스 +5, 대화 중 조언 제공 |
| **소망** (Hopeful) | Ch.6 후반 | 게임 끝까지 | hopeful_trust | 믿음 보너스 +5, 의심의 성에서 핵심 역할 |

동행자 관계 스탯은 대화 선택에 따라 변화하며, 높을수록 동행자가 위기 상황에서 더 적극적으로 도움을 준다.

```ink
// 신실과의 관계가 높으면 허영의 시장에서 추가 대사
{faithful_trust >= 60:
    # SPEAKER: 신실
    # EMOTION: resolute
    "형제여, 두려워 마시오. 우리가 어떤 고난을 받더라도—"
    "이 시장의 어떤 것도 우리의 참된 보화를 빼앗지 못할 것이오."
}
```

## 4. 전투/챌린지 시스템 (Combat/Challenge System)

본 게임은 전통적 RPG 전투가 아닌, **서사에 봉사하는 맥락적 챌린지**를 사용한다. 각 챌린지는 해당 장소의 우화적 의미를 게임플레이로 체현한다.

### 4.1 QTE 전투: 아폴리온과의 싸움

**장소**: 겸손의 골짜기 (Ch.4)
**유형**: 리듬 기반 QTE + 선택지 혼합
**소요 시간**: 약 5~7분

#### 전투 흐름

```
[1단계: 아폴리온의 위협]
  아폴리온이 나타나 크리스천에게 항복을 요구한다.
  → 대화 선택: 항복 거부 (필수) — 용기에 따라 대사 변형

[2단계: 화살 공격]
  아폴리온이 "비 쏟듯이 화살을 쏜다" (원작 표현)
  → QTE: 화면에 나타나는 화살 방향에 맞춰 방패 방향키 입력
  → 4~6회 연속, 점점 빨라짐
  → 실패 시: 크리스천이 부상 (화면 붉어짐), 재시도 가능

[3단계: 격투]
  아폴리온과 근접 전투. "반나절 동안 싸웠다" (원작)
  → QTE: 타이밍에 맞춰 공격(J)/방어(K) 입력
  → 성공률에 따라 크리스천의 체력 바 변화
  → 용기 ≥ 50이면 추가 공격 옵션 "성령의 검으로 베다" 등장

[4단계: 위기]
  아폴리온이 크리스천을 넘어뜨리고 검을 빼앗는다.
  → 자동 연출: 크리스천 쓰러짐, 아폴리온이 치명타를 가하려 함
  → 선택지 (시간 제한 5초):
    * "주여, 도와주소서!" (믿음 기반) → 성공
    * [아무 입력 없음] → 실패 후 재시도

[5단계: 역전]
  크리스천이 검을 다시 잡고 아폴리온에게 치명적 일격.
  → QTE: 최종 일격 타이밍 (단일 입력)
  → "아폴리온이 날개를 펴고 날아가 다시 보이지 않았다"
  → 승리 연출: 믿음 +15, 용기 +20, 생명의 나뭇잎으로 치유

[보너스]
  전투 중 QTE 성공률 90% 이상 → 성경 구절 카드 "엡 6:16" 획득
```

#### QTE 입력 매핑

| 플랫폼 | 공격 | 방어 | 특수 (기도) |
|--------|------|------|------------|
| **PC** | J 또는 좌클릭 | K 또는 우클릭 | Space |
| **모바일** | 우측 탭 | 좌측 탭 | 두 손가락 탭 |
| **컨트롤러** | A/X | B/O | Y/△ |

### 4.2 은닉/인내 챌린지: 사망의 음침한 골짜기

**장소**: 사망의 음침한 골짜기 (Ch.4)
**유형**: 인내 챌린지 (타이밍 + 선택)
**소요 시간**: 약 4~6분

#### 챌린지 흐름

```
[구간 1: 입구의 어둠]
  화면이 점차 어두워짐. 배경에 괴물 소리, 속삭임.
  → 플레이어 행동: "앞으로 계속 걷기" 버튼을 꾹 누르고 있기
  → 누르지 않으면 크리스천이 멈추고 두려움 게이지 상승
  → 두려움 게이지 MAX → 되돌아가는 유혹 선택지 등장

[구간 2: 좁은 길]
  좌우에 수렁과 구덩이. 길이 매우 좁음.
  → 좌우 방향키로 균형 유지 (기울어지는 바 안정화)
  → 너무 한쪽으로 치우치면 "미끄러짐" → 체력 감소
  → 시 23:4 "사망의 음침한 골짜기로 다닐지라도" 자동 재생

[구간 3: 마귀의 속삭임]
  화면에 검은 텍스트로 불경한 생각이 나타남 (원작: 마귀가 귀에 속삭임)
  → 선택지: 각 속삭임에 대응하는 말씀 선택
    * "하나님은 없다" → 대응: "어리석은 자는 마음에 이르기를 하나님이 없다 하도다" (시 14:1)
    * "네 죄는 용서받지 못한다" → 대응: "모든 죄와 훼방은 사함을 받되" (마 12:31)
    * "돌아가라, 여기서 죽으리라" → 대응: "여호와는 나의 목자시니 내게 부족함이 없으리로다" (시 23:1)
  → 올바른 말씀 선택 시 어둠이 약간 걷힘, 지혜 +3
  → 틀린 선택 시 어둠 심화, 두려움 게이지 상승

[구간 4: 새벽]
  "그때 아침이 밝았다." 화면이 점차 밝아짐.
  → 자동 연출: 크리스천이 지나온 길을 돌아보며 놀람
  → 결과: 믿음 +15, 용기 +15, 지혜 +8
```

### 4.3 대화 전투: 허영의 시장 재판

**장소**: 허영의 시장 (Ch.6)
**유형**: 대화 전투 (논쟁/변호)
**소요 시간**: 약 6~8분

#### 재판 구조

| 단계 | 내용 | 플레이어 역할 |
|------|------|-------------|
| **기소** | 시기, 미신, 아첨이 신실을 고발 | 목격 (자동) |
| **증인 심문** | 세 증인의 거짓 증언 | 반응 선택 (분노/침묵/기도) |
| **신실의 변호** | 신실이 스스로를 변호 | 선택지로 변호 내용 결정 |
| **판결** | 증선 재판관의 유죄 판결 | 반응 선택 (항의/수용) |
| **처형** | 신실의 순교 | 자동 연출 + 감정 선택 |

```ink
=== vanity_fair_trial ===
```

### 4.4 특수 챌린지: 의심의 성 탈출

**장소**: 의심의 성 (Ch.7)
**유형**: 퍼즐 + 대화 (약속의 열쇠 발견)
**소요 시간**: 약 5~7분

#### 챌린지 흐름

```
[1일차]
  거인 절망이 순례자들을 발견하고 지하 감옥에 가둠.
  → 환경 탐색: 감옥 내 인터랙티브 포인트 탐색
  → 절망의 말에 대응하는 선택지 (자살 유혹 거부)

[2일차]
  거인 절망이 다시 와서 고문하고 자살을 권유.
  → 소심부인이 등장하여 더 잔혹한 방법 제안
  → 소망이 크리스천을 격려: "기억하시오, 형제여!"
  → 선택지: 과거 여정의 기억을 되살림

[3일차 — 핵심 퍼즐]
  "토요일 밤, 자정 무렵" — 소망과 크리스천이 기도를 시작한다.
  → 크리스천이 갑자기 외침: "내가 어찌 이리 어리석은가!"
  → 퍼즐: 플레이어가 크리스천의 품속에서 "약속"이라는 열쇠를 찾아야 함
    * 인벤토리 화면에서 두루마리 → 열쇠 아이콘 발견
    * 또는 대화에서 "히브리서 13:5 — 내가 결코 너를 버리지 아니하리라" 선택
  → 열쇠로 감옥 문 열기 → 안뜰 문 열기 → 성문 열기

[탈출]
  거인 절망이 쫓아오나 약해짐 (발작)
  → 탈출 성공: 믿음 +20, 용기 +10, 지혜 +15
  → 성문 밖에 "다시는 이 길로 들어오지 말라"는 경고판 세움
```

### 4.5 챌린지 난이도 설정

모든 챌린지는 접근성을 위해 난이도 선택을 제공한다:

| 난이도 | QTE 속도 | 시간 제한 | 실패 패널티 | 대상 |
|--------|---------|----------|-----------|------|
| **이야기 모드** | 매우 느림 | 없음 | 없음 (자동 성공) | 서사만 즐기고 싶은 플레이어 |
| **순례자 모드** (기본) | 보통 | 넉넉함 | 경미 (재시도 즉시) | 일반 플레이어 |
| **전사 모드** | 빠름 | 촉박함 | 체력 감소, 스탯 패널티 | 도전을 원하는 플레이어 |

## 5. 수집 시스템 (Collection System)

### 5.1 성경 구절 카드 (Bible Verse Cards)

각 장소에서 관련 성경 구절을 카드 형태로 발견하고 수집할 수 있다. 총 52장으로 구성.

#### 카드 설계

```
┌─────────────────────────────┐
│  ═══════════════════════    │
│  [성경 구절 카드]            │
│  ═══════════════════════    │
│                             │
│  📖 마태복음 7:14            │
│                             │
│  "좁은 문으로 들어가라       │
│   생명으로 인도하는 문은     │
│   좁고 길이 협착하여         │
│   찾는 이가 적음이라"        │
│                             │
│  ─────────────────────      │
│  발견 장소: 좁은 문          │
│  카테고리: 구원의 길          │
│  ═══════════════════════    │
└─────────────────────────────┘
```

#### 카드 카테고리와 장소별 배분

| 카테고리 | 카드 수 | 관련 장소 | 보상 |
|---------|--------|----------|------|
| **죄와 회개** | 8장 | 멸망의 도시, 절망의 늪, 시내산 | 대화 선택지 1개 해금 |
| **구원의 길** | 8장 | 좁은 문, 해석자의 집, 십자가 | 대화 선택지 1개 해금 |
| **신앙의 성장** | 8장 | 어려움의 언덕, 궁전, 겸손의 골짜기 | 여정 일지 보너스 엔트리 |
| **영적 전쟁** | 7장 | 겸손의 골짜기, 사망의 골짜기, 허영의 시장 | QTE 보너스 (시간 여유) |
| **경고와 교훈** | 7장 | 허영의 시장, 샛길 초원, 의심의 성 | 캐릭터 도감 확장 |
| **소망과 인내** | 7장 | 기쁨의 산, 마법의 땅, 블라의 나라 | 특별 내레이션 해금 |
| **영광과 완성** | 7장 | 사망의 강, 천성 | 엔딩 보너스 연출 |

#### 카드 발견 방법

| 방법 | 설명 | 비율 |
|------|------|------|
| **메인 스토리** | 스토리 진행 중 자동 획득 | 약 40% (21장) |
| **탐험 포인트** | 배경의 빛나는 포인트 클릭 | 약 35% (18장) |
| **대화 선택** | 특정 대화 선택지를 골랐을 때만 획득 | 약 15% (8장) |
| **챌린지 보너스** | 챌린지 고성적 달성 시 | 약 10% (5장) |

### 5.2 캐릭터 도감 (Character Encyclopedia)

만난 모든 캐릭터의 정보가 자동으로 기록된다. 총 40여 명.

#### 도감 항목 구성

| 항목 | 내용 |
|------|------|
| **이름** | 한국어명 / 영문 원제 (예: 전도자 / Evangelist) |
| **초상화** | 캐릭터 일러스트 (처음에는 실루엣, 만남 후 해금) |
| **상징** | 이 캐릭터가 우화적으로 상징하는 것 |
| **핵심 대사** | 가장 인상적인 대사 1~2개 |
| **만남 장소** | 어느 장소에서 만났는지 |
| **관련 성경** | 이 캐릭터와 관련된 성경 구절 |
| **해석자 메모** | 만남 이후 해금되는 심화 해설 (선택적 읽기) |

#### 도감 해금 단계

| 단계 | 조건 | 해금 내용 |
|------|------|----------|
| **1단계** | 캐릭터와 첫 만남 | 실루엣 → 초상화 전환, 이름/만남 장소 |
| **2단계** | 해당 장면 완료 | 핵심 대사, 상징, 관련 성경 |
| **3단계** | 관련 성경 카드 수집 | 해석자 메모 (심화 해설) |

### 5.3 여정 일지 (Journey Journal)

플레이어의 선택과 여정이 자동으로 기록되는 일지. 양피지 질감의 UI에 필기체로 표시.

#### 일지 구성

| 섹션 | 내용 |
|------|------|
| **경로 기록** | 방문한 장소와 순서, 각 장소에서 보낸 시간 |
| **선택 기록** | 주요 선택지에서 무엇을 골랐는지 (되돌아보기용) |
| **스탯 변화 그래프** | 믿음/용기/지혜의 변화를 선형 그래프로 표시 |
| **크리스천의 독백** | 각 장소를 떠날 때 자동 생성되는 짧은 감상 (선택에 따라 내용 변화) |
| **짐 무게 기록** | 짐의 변화를 시각적으로 추적 (십자가에서 0이 되는 순간 강조) |

### 5.4 수집 보상 체계

| 수집 마일스톤 | 조건 | 보상 |
|-------------|------|------|
| **첫 발견** | 성경 카드 1장 | 수집 시스템 튜토리얼 해금 |
| **탐구자** | 성경 카드 10장 | 갤러리에 "원작 삽화" 1장 해금 |
| **열심** | 성경 카드 25장 | 갤러리에 "원작 삽화" 3장 해금 |
| **완전무장** | 성경 카드 52장 (전체) | 특별 엔딩 크레딧 + "번연의 서문" 해금 |
| **만남의 기록** | 캐릭터 20명 도감 완성 | 타이틀 화면 변경 (동행자들이 함께 걷는 모습) |
| **순례 완수** | 여정 일지 전체 장소 기록 | 2회차 "해석자 해설 모드" 해금 |

| **짐 게이지** | 초상화 아래 | 십자가 전까지만 표시. 무거울수록 붉은색, 가벼울수록 금색 |

**HUD 최소화 모드**: 대화 및 컷신 중에는 스탯 바와 짐 게이지가 자동으로 페이드아웃되어 서사 몰입을 방해하지 않는다. 스탯 변화 발생 시에만 해당 바가 잠깐 나타났다 사라진다.

| 대화 UI 요소 | 설명 |
|-------------|------|
| **텍스트 박스** | 하단 1/3 영역. 양피지 질감 반투명 배경. 둥근 모서리 |
| **화자 이름** | 텍스트 박스 좌상단에 작은 태그로 표시 |
| **초상화** | 텍스트 박스 좌측에 소형, 화면 중앙에 대형 (상황에 따라) |
| **텍스트 출력** | 타자기 효과 (글자가 하나씩 나타남). 클릭으로 즉시 완성 |
| **선택지 버튼** | 텍스트 박스 아래에 1~4개 배치. 양피지 버튼 스타일 |
| **스탯 변화 미리보기** | 선택지에 마우스 호버 시 예상 변화 표시 (설정에서 끄기 가능) |
| **스탯 게이팅 표시** | 스탯 부족 시 선택지가 회색 + 자물쇠 아이콘 + 필요 스탯 표시 |

**대화 텍스트 스타일 규칙**:
- 화자(NPC) 대사: 기본 세리프체, 다크 브라운

## 7. 오디오 설계 (Audio Design)

### 7.1 BGM 전략

BGM은 장소의 감정 톤과 우화적 의미에 맞춰 설계한다. 1인 개발 현실을 고려하여 AI 작곡 도구와 로열티 프리 라이브러리를 활용한다.

#### 장소별 BGM 매핑

| 장소 | 감정 톤 | 악기/스타일 | 템포 | 키 |
|------|---------|-----------|------|-----|
| **멸망의 도시** | 불안, 긴박 | 현악 트레몰로 + 낮은 드론 | 느림 → 빨라짐 | D minor |
| **들판 (전도자 만남)** | 희미한 희망 | 솔로 오보에 + 약한 현악 | 보통 | G minor → G major |
| **절망의 늪** | 절망, 무거움 | 첼로 솔로 + 짧은 침묵 | 매우 느림 | C minor |
| **시내산** | 두려움, 위압 | 금관 + 팀파니 트레몰로 | 빠름(격렬) | E♭ minor |
| **좁은 문** | 안도, 경건 | 합창풍 패드 + 하프 아르페지오 | 느림 | D major |
| **해석자의 집** | 경이, 배움 | 쳄발로 + 현악 4중주 | 보통 | B♭ major |
| **십자가** | 카타르시스, 감격 | 오르간 + 풀 오케스트라 빌드업 → 합창 | 느림 → 고조 | C major |
| **어려움의 언덕** | 결의, 고투 | 스네어 리듬 + 현악 상행 | 빠름 | A minor |
| **정자** | 평화, 나태 | 목관 자장가풍 | 매우 느림 | F major |
| **아름다운 궁전** | 따뜻함, 교제 | 류트 + 리코더 + 현악 | 보통 | G major |
| **겸손의 골짜기** | 긴장, 전투 | 전투 드럼 + 금관 팡파르 | 빠름 | D minor |
| **아폴리온 전투** | 격렬, 영적 전쟁 | 풀 오케스트라 전투곡 + 합창 | 매우 빠름 | C minor |
| **사망의 음침한 골짜기** | 공포, 어둠 | 미니멀: 저음 드론 + 불규칙 타격음 | 불규칙 | 무조성 |
| **신실 동행** | 우정, 여행 | 어쿠스틱 기타 + 플루트 | 경쾌 | A major |
| **허영의 시장** | 혼란, 소란 | 중세 페어 음악(허디거디, 피들) + 군중 소음 | 빠름 | F major (아이러니) |
| **재판/순교** | 비극, 숭고 | 솔로 바이올린 → 합창 레퀴엠 | 느림 | E minor → C major |
| **안일의 평지** | 유혹, 게으름 | 달콤한 하프 + 졸음 유도 패드 | 매우 느림 | A♭ major |
| **의심의 성** | 절망, 갇힘 | 금속성 타격음 + 저음 현악 | 매우 느림 | B minor |
| **기쁨의 산** | 평화, 조망 | 팬플루트 + 현악 + 새소리 | 보통 | D major |
| **마법의 땅** | 졸음, 위험 | 몽환적 신스패드 + 딜레이 하프 | 느림 | 모호한 조성 |
| **블라의 나라** | 순수한 기쁨 | 현악 세레나데 + 새소리 + 바람 | 보통 | G major |
| **사망의 강** | 경외, 두려움 | 물 소리 + 저음 합창 → 빛의 합창 | 느림 → 고조 | A minor → C major |
| **천성** | 절대 기쁨, 영광 | 풀 오케스트라 + 대합창 + 나팔 팡파르 | 장엄 | C major |

#### BGM 전환 규칙

| 전환 유형 | 방법 | 시간 |
|---------|------|------|
| **장소 이동** | 크로스페이드 | 2~3초 |
| **감정 변화** | 같은 곡 내 레이어 추가/제거 | 1~2초 |
| **전투 돌입** | 하드컷 (드럼 인트로) | 즉시 |
| **전투 종료** | 빠른 디크레셴도 → 평화 테마 | 3~4초 |
| **CG/컷신** | 전용 BGM으로 교체 | 1초 |

### 7.2 SFX 카테고리

| 카테고리 | 예시 | 필요 수량 |
|---------|------|----------|
| **UI** | 버튼 클릭, 페이지 넘김, 메뉴 열기/닫기 | 8~10개 |
| **이동** | 발걸음 (흙/돌/물/나무), 짐 삐걱거림, 문 열기 | 12~15개 |
| **대화** | 텍스트 타이핑음, 선택 확인음, 감정 강조음 | 6~8개 |
| **전투** | 검 부딪힘, 방패 막기, 화살 날아감, 타격, 기도 효과 | 15~20개 |
| **환경** | 바람, 빗소리, 천둥, 새소리, 물소리, 불 타는 소리 | 15~20개 |
| **시스템** | 스탯 변화, 카드 획득, 도감 해금, 저장 완료 | 6~8개 |
| **특수** | 짐 떨어짐, 짐 끈 끊어짐, 십자가 빛, 나팔 소리, 천사 합창 | 8~10개 |

### 7.3 앰비언트 사운드

각 장소에 고유한 앰비언트 루프가 깔린다. BGM과 별도 레이어로 재생.

| 장소 | 앰비언트 |
|------|---------|
| 멸망의 도시 | 먼 곳의 불길 소리, 아이 울음, 개 짖음 |
| 들판 | 바람, 풀잎 소리, 먼 새소리 |
| 절망의 늪 | 진흙 소리, 물 거품, 무거운 숨소리 |
| 좁은 문 | 먼 화살 소리 (바알세불의 성), 문 삐걱 |
| 해석자의 집 | 촛불 타닥거림, 먼 시계 소리 |
| 십자가 | 바람이 멈추고 고요 → 새소리 시작 |
| 궁전 | 벽난로, 식기 소리, 먼 합창 |
| 겸손의 골짜기 | 바람, 풀잎, 갑자기 정적 (아폴리온 등장 전) |
| 사망의 골짜기 | 괴물 울음, 속삭임, 불규칙한 금속 소리 |
| 허영의 시장 | 군중 소란, 상인 호객, 음악, 웃음소리 |
| 의심의 성 | 빗소리, 쇠사슬 딸깍, 거인의 발소리 |
| 사망의 강 | 깊은 물소리, 바람, 먼 곳의 노래 |
| 천성 | 합창, 나팔, 종소리, 기쁨의 함성 |

### 7.4 1인 개발자를 위한 오디오 제작 전략

| 도구/리소스 | 용도 | 비용 |
|-----------|------|------|
| **Suno AI / Udio** | BGM 초안 생성 (스타일/감정 지정 프롬프트) | 월 $10~30 |
| **AIVA** | 클래식 스타일 BGM 작곡 (오케스트라/합창) | 월 $15 |
| **Freesound.org** | CC0 SFX, 앰비언트 루프 | 무료 |
| **Zapsplat** | 고품질 SFX 라이브러리 | 무료/연 $36 |
| **Audacity** | SFX 편집, 노이즈 제거, 루프 편집 | 무료 |
| **FMOD / Wwise (무료 티어)** | Unity 오디오 미들웨어 (크로스페이드, 레이어링) | 무료 (인디) |

**워크플로우**:
1. 각 장소의 감정 톤과 키워드를 AI 작곡 도구에 입력
2. 생성된 3~5개 후보 중 가장 적합한 것 선택
3. Audacity에서 루프 포인트 편집, 인트로/아웃트로 조정
4. FMOD에서 레이어링 설정 (앰비언트 + BGM + SFX 동시 재생)
