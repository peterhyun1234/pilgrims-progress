# 01. 게임 컨셉 & 비전 (Game Concept & Vision)

> **프로젝트명**: 천로역정 — 순례자의 여정 (The Pilgrim's Progress)
> **장르**: 인터랙티브 내러티브 어드벤처 (Interactive Narrative Adventure)
> **엔진**: Unity 6 + Ink (서사 시스템)
> **플랫폼**: PC (Windows/macOS) → Mobile (iOS/Android) → Steam
> **개발 체제**: 1인 개발
> **최종 수정**: 2026-03-15

---

## 1. 엘리베이터 피치

> *300년간 전 세계 2억 부 이상 읽힌 존 번연의 명작 '천로역정'을*
> *당신이 직접 걸어가는 인터랙티브 어드벤처로 경험하세요.*
> *모든 선택이 크리스천의 운명을 바꾸고, 모든 만남이 당신의 마음을 울립니다.*

**한 줄 요약**: 선택과 결과가 있는 서사 중심 어드벤처 게임. 크리스천의 순례길을 직접 걸으며, 각 장소에서의 대화·결단·전투를 통해 복음의 메시지를 체험한다.

---

## 2. 핵심 경험 목표 (Core Experience Goals)

### 크리스천(신자)을 위한 경험
- 익숙한 성경 이야기를 **새로운 미디어**로 재발견하는 감동
- 각 알레고리의 **실제 삶에서의 의미**를 게임 선택지를 통해 체감
- 원작의 깊은 신학적 메시지를 **인터랙티브하게** 탐구

### 넌크리스천(비신자·탐구자)을 위한 경험
- 종교적 선입견 없이 **좋은 이야기**로서 천로역정에 입문
- 게임 속 선택과 결과를 통해 **자연스럽게** 복음의 핵심 질문과 마주침
- "내 등에도 짐이 있지 않은가?"라는 **보편적 질문**에서 시작하는 공감

### 게임으로서의 경험
- **긴장감 있는 선택**: 모든 대화와 갈림길에서 의미 있는 결단을 내림
- **탐험의 즐거움**: 각 장소의 고유한 분위기, 숨겨진 대화, 성경 구절 수집
- **성장의 보람**: 믿음·용기·지혜 스탯이 선택에 따라 변화하며 스토리에 영향
- **감정적 몰입**: 아폴리온과의 전투, 신실의 순교, 사망의 강 건너기 등 강렬한 순간들

## 3. 장르 결정: 인터랙티브 내러티브 어드벤처

### 왜 이 장르인가?

| 고려 사항 | 결정 근거 |
|-----------|----------|
| **원작의 본질** | 천로역정은 대화와 서사가 핵심. 액션보다 이야기가 중심이어야 함 |
| **1인 개발 실현성** | VN/내러티브 장르는 아트 에셋 최소, 코드 복잡도 낮음 |
| **타겟 확장성** | 게임을 잘 모르는 크리스천도 접근 가능한 낮은 진입 장벽 |
| **감정 전달력** | 텍스트 + 일러스트 + 음악 조합이 감정 전달에 가장 효과적 |
| **콘텐츠 확장** | 챕터 단위 추가가 용이 (Ink 파일만 추가) |

### 하이브리드 요소

순수 비주얼 노벨이 아닌, **간단한 게임플레이 요소**를 결합합니다:

| 요소 | 구현 방식 | 해당 장면 |
|------|----------|----------|
| **선택 & 분기** | Ink 기반 대화 선택지 | 모든 장소 |
| **탐험** | 2D 맵 위 캐릭터 이동 (포인트 앤 클릭) | 장소 간 이동, 숨겨진 요소 |
| **스탯 기반 결과** | 믿음/용기/지혜에 따라 대화 옵션/결과 변화 | 주요 갈림길 |
| **QTE/미니게임** | 간단한 입력으로 전투/도전 표현 | 아폴리온 전투, 사망의 골짜기 |
| **수집** | 성경 구절 카드, 캐릭터 도감 | 전 구간 |

## 4. 아트 디렉션

### 스타일: **따뜻한 수채화/동화책 (Storybook Watercolor)**

천로역정의 본질은 '꿈속의 우화'입니다. 아트 스타일도 이 꿈결 같은 우화적 분위기를 시각적으로 전달해야 합니다.

| 요소 | 방향 |
|------|------|
| **전체 톤** | 따뜻한 수채화풍. 빛과 어둠의 대비가 극적 |
| **배경** | 넓은 붓 터치, 부드러운 그라데이션, 동화책 일러스트 느낌 |
| **캐릭터** | 반 실사 일러스트. 표정이 풍부한 상반신 초상화 |
| **빛의 장소** (궁전, 블라, 천성) | 따뜻한 금빛/호박색 톤, 빛 입자 |
| **어둠의 장소** (골짜기, 의심의 성) | 차가운 남색/보라 톤, 안개/그림자 |
| **UI** | 양피지/고서적 질감. 클래식하되 깔끔한 타이포그래피 |

### 컬러 팔레트

```
[빛의 팔레트 — 구원, 교제, 희망]
Primary:   #C8956C (따뜻한 황토)
Secondary: #F5E6D3 (밀크)
Accent:    #D4A853 (금빛)
Text:      #3B2F2F (다크 브라운)

[어둠의 팔레트 — 시련, 절망, 유혹]
Primary:   #1E2A3A (깊은 남색)
Secondary: #2D1B4E (보라 어둠)
Accent:    #8B3A3A (어두운 적색)
Text:      #E8E0D0 (바랜 양피지)

[전투/긴장 팔레트]
Primary:   #4A1C1C (핏빛 적갈색)
Accent:    #FF6B35 (불꽃 오렌지)
```

### 에셋 생산 전략

**AI 생성 + 수작업 정제** 방식:
1. Midjourney/Stable Diffusion으로 배경·캐릭터 초안 생성
2. Photoshop/Krita로 스타일 통일 및 정제
3. 게임 내 연출(파티클, 조명)은 Unity에서 처리

> 상세 파이프라인은 `05-asset-pipeline.md` 참조

## 5. 타겟 유저

### 핵심 타겟

| 세그먼트 | 비중 | 프로필 |
|----------|------|--------|
| **크리스천 게이머** | 40% | 15~35세, 교회 청년부/대학부, 게임에 익숙, 신앙 콘텐츠에 갈증 |
| **크리스천 비게이머** | 25% | 20~50세, 게임 경험 적음, 좋은 신앙 콘텐츠 찾는 중, 쉬운 조작 필요 |
| **서사 게임 팬** | 20% | 15~40세, 종교 무관, 좋은 이야기면 OK, Undertale/Celeste류 팬 |
| **탐구자/구도자** | 15% | 전 연령, 기독교에 관심 있으나 교회엔 안 감, 게임으로 먼저 접근 |

### 이용 등급
- **전 연령 가능 (E / 전체 이용가)**
- 폭력: 전투 장면 있으나 피/고어 없음 (추상적 표현)
- 종교: 기독교 알레고리 명시적이나 강압적이지 않음

| 모델 | 가격 | 포함 내용 |
|------|------|----------|

- 사역 목적이 강하다면, 무료 데모만으로도 복음의 핵심(죄의 자각 → 십자가)을 전달 가능

## 7. MVP 범위 결정

### MVP = 무료 데모: **멸망의 도시 → 십자가** (Chapter 01~02)

| 항목 | 내용 |
|------|------|
| **포함 장소** | 멸망의 도시, 들판, 절망의 늪, 세상지혜씨 만남, 시내산, 좁은 문, 해석자의 집 (7개 방), 십자가/무덤 |
| **포함 캐릭터** | 크리스천, 전도자, 완고, 유연, 도움, 세상지혜씨, 선의, 해석자, 빛나는 자 3인 |
| **예상 플레이타임** | 40~60분 |
| **선택지/분기** | 15~20개 주요 선택, 3가지 엔딩 변형 |
| **스탯 시스템** | 믿음/용기/지혜 3개 스탯 도입 |
| **엔딩** | 십자가에서 짐이 떨어지는 장면 (카타르시스) |

**왜 이 범위인가?**
- 원작에서 가장 강력한 서사 아크: **죄의 자각 → 방황 → 은혜 → 구원**
- 복음의 핵심 메시지를 완결된 형태로 전달
- 40~60분은 무료 데모로 적절한 분량
- "짐이 떨어지는 순간"이 강력한 클리프행어 → 정식판 구매 동기

> 상세 MVP 스펙은 `04-mvp-specification.md` 참조

## 9. 핵심 시스템 개요

### 9.1 서사 시스템 (Ink)
- 모든 대화, 선택지, 분기는 `.ink` 파일로 관리
- 스탯 변화, 아이템 획득도 Ink 변수로 처리
- 챕터별 독립 Ink 파일 → 콘텐츠 추가 용이

### 9.2 스탯 시스템
```
[크리스천의 세 가지 덕목]

믿음 (Faith)   — 하나님에 대한 신뢰. 두려운 선택에서 나아갈 때 증가
용기 (Courage)  — 시련 앞에서의 담대함. 위험을 감수할 때 증가
지혜 (Wisdom)   — 영적 분별력. 올바른 판단을 내릴 때 증가

* 스탯은 선택에 따라 ±5~15 단위로 변동
* 특정 스탯이 임계값 이상이면 새로운 대화 옵션 해금
* 스탯 조합에 따라 일부 장면의 결과가 달라짐
```

### 9.3 수집 시스템
- **성경 구절 카드**: 각 장소에서 관련 구절을 발견/수집. 도감에 저장
- **캐릭터 도감**: 만난 인물의 이름, 상징, 핵심 대사 기록
- **여정 일지**: 지나온 장소와 선택의 기록을 자동 저장

### 9.4 연출 시스템
- **화면 전환**: 장소 이동 시 수채화 번짐 효과 (Dissolve)
- **감정 연출**: 대화 중 화면 흔들림, 빛 변화, 음악 전환
- **전투**: QTE 기반 — 타이밍에 맞춰 버튼 입력으로 공격/방어

## 10. 경쟁작 분석 & 차별점

| 게임 | 유사점 | 천로역정의 차별점 |
|------|--------|-----------------|
| **Undertale** | 선택과 결과, 강한 서사, 도덕적 딜레마 | 실제 고전 문학 기반, 성경적 세계관, 더 깊은 우화적 의미 |
| **Coffee Talk** | 대화 중심, 따뜻한 분위기 | 능동적 선택/분기, 전투/탐험 요소, 명확한 서사 아크 |
| **Celeste** | 개인의 성장 서사, 인디 감성 | 대화/선택 중심(액션 아님), 보편적 고전 서사 |
| **Bible Adventures (구작)** | 성경 기반 | 현대적 게임 디자인, 강압적이지 않은 메시지 전달, 높은 예술적 품질 |

### 이 게임만의 USP (Unique Selling Point)
1. **300년 검증된 서사**: 새로 쓴 이야기가 아닌, 역사가 증명한 명작의 인터랙티브화
2. **복음의 체험**: 교리를 '가르치는' 것이 아니라, 크리스천의 발걸음으로 '걸어보는' 것
3. **보편적 주제**: "등에 진 짐"은 죄의식이든 트라우마든 번아웃이든, 누구나 공감하는 인간 조건
4. **무료 복음**: 데모만으로 완결된 복음 메시지 전달 (멸망의 도시 → 십자가)

## 11. 프로젝트 비전 선언

> **"좋은 게임은 플레이어에게 즐거움을 주고,**
> **위대한 게임은 플레이어의 마음을 바꿉니다."**
>
> 천로역정은 300년간 수억 명의 마음을 바꿔온 이야기입니다.
> 이 게임은 그 이야기를, 21세기의 언어로 — 인터랙티브 미디어로 — 다시 전합니다.
>
> 크리스천이든 아니든, 이 게임을 끝낸 사람이 "나도 등에 짐을 지고 있었구나"라고 느끼고,
> "그 짐을 내려놓을 곳이 있다"는 것을 알게 되기를 바랍니다.
>
> 화려한 그래픽이나 복잡한 시스템이 아니라,
> **진심 어린 이야기와 의미 있는 선택**이 이 게임의 전부입니다.
