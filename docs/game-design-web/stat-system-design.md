# 스탯 시스템 상세 설계서

> 천로역정: 순례자의 여정 — Web Reboot  
> 버전: 1.0 | 최종 수정: 2026-03-19

---

## 1. 설계 철학

천로역정의 스탯 시스템은 단순 RPG 수치가 아니라 **영적 순례의 내면 상태를 게임 메카닉으로 번역**한 것이다.

- **스탯은 레벨이 아니라 상태다** — 올라가기만 하는 것이 아니라 선택에 따라 오르내린다
- **모든 스탯 변화에는 서사적 의미가 있다** — 숫자가 아니라 이야기가 변한다
- **게임 오버 없음** — 스탯이 0이 되어도 진행 가능하며, 오히려 더 깊은 서사 경로가 열린다
- **Burden은 유일한 "해방" 스탯** — 다른 스탯은 끝까지 변동하지만, Burden만 Ch6에서 영구적으로 0이 된다

참조: Undertale의 LV/EXP(폭력이 수치에 새겨짐), Celeste의 어시스트 모드(실패를 벌하지 않음)

### 2.2 각 스탯의 게임플레이 영향

#### Faith (믿음)

| 범위 | 효과 |
|------|------|
| 0–19 | 유혹 대화에서 거부 선택지 비활성. NPC가 걱정하는 대사 추가. 전도자 자동 등장 이벤트 발동 |
| 20–39 | 기본 대화만 가능. 일부 성경 카드 힌트가 보이지 않음 |
| 40–59 | 표준 플레이. 대부분의 대화 선택지 사용 가능 |
| 60–79 | 고급 대화 선택지 해금 (성경 인용, 영적 분별). NPC 반응이 더 깊어짐 |
| 80–100 | 특별 연출 해금 (Ch6 십자가 확장 연출, 빛나는 자들 추가 대화). QTE 난이도 감소 (타이밍 윈도우 +20%) |

#### Courage (용기)

| 범위 | 효과 |
|------|------|
| 0–19 | 위험 지역 진입 시 속도 추가 감소 (-15%). 공포 이벤트에서 화면 흔들림 강화. 사자 앞에서 자동 멈춤 |
| 20–39 | 전투 QTE 타이밍 윈도우 기본값. 일부 용기 대화 비활성 |
| 40–59 | 표준 플레이. 전투 성과 보통 |
| 60–79 | QTE 타이밍 윈도우 +15%. 위험 지역 속도 감소 없음. 용기 있는 대사 선택 가능 |
| 80–100 | 보스전 특별 대사 해금. "담대한 자" 칭호. 아볼론 전투 최종 단계 단축 |

#### Wisdom (지혜)

| 범위 | 효과 |
|------|------|
| 0–19 | 퍼즐 힌트 없음. 함정 경고 없음. 수다(Talkative) 같은 NPC를 분별하지 못함 |
| 20–39 | 기본 퍼즐 힌트 1개. 일부 숨겨진 선택지 비활성 |
| 40–59 | 표준 플레이. 퍼즐 힌트 2개. 대부분의 숨겨진 경로 접근 가능 |
| 60–79 | 퍼즐 힌트 3개 + 추가 해법. NPC의 본성을 미리 분별하는 내면 독백 |
| 80–100 | 모든 숨겨진 선택지 해금. 무지(Ignorance)와의 대화에서 추가 논증. 의심의 성 퍼즐 단축 |

#### Burden (짐)

| 범위 | 속도 배율 | 비주얼 | 사운드 |
|------|----------|--------|--------|
| 81–100 | ×0.50 (50% 감소) | 거대한 짐 + 땀 파티클 + 어두운 옷 | 무거운 발소리, 신음 |
| 61–80 | ×0.60 | 큰 짐 + 약간 구부러진 자세 | 무거운 발소리 |
| 31–60 | ×0.75 | 중간 짐 | 보통 발소리 |
| 1–30 | ×0.90 | 작은 짐 | 가벼운 발소리 |
| 0 | ×1.00 | 짐 없음 + 밝은 옷 + 빛 오라 | 경쾌한 발소리 |

**속도 공식:**
```
actualSpeed = BASE_SPEED × (1.0 - burden / 100 × 0.5)
```
- `BASE_SPEED = 80` (px/sec, 320x180 내부 해상도 기준)
- burden 80 → `80 × 0.60 = 48 px/sec`
- burden 0 → `80 × 1.00 = 80 px/sec`

## 3. 숨겨진 스탯 (Hidden Stats)

플레이어에게 직접 표시되지 않지만 서사와 연출에 영향을 미치는 내부 수치.

### 3.1 Relationship (NPC 호감도)

각 주요 NPC와의 관계 수치. 대화 선택에 따라 변동.

| NPC | 초기값 | 범위 | 영향 |
|-----|--------|------|------|
| Evangelist (전도자) | 50 | 0–100 | 교정 이벤트 대사 변화. 높으면 더 따뜻한 격려, 낮으면 엄격한 경고 |
| Good-will (선의) | 0 | 0–100 | 좁은 문 통과 연출 변화 |
| Interpreter (해석자) | 0 | 0–100 | 해석자의 집 추가 방 해금 (호감 60+) |
| Faithful (신실) | 0 | 0–100 | Ch10 재판 장면 대사 분기, 순교 연출 강도 |
| Hopeful (소망) | 0 | 0–100 | Ch12 요단강 장면 격려 대사 변화 |

**변동 규칙:**
- 해당 NPC와의 대화에서 공감/존중 선택 → +5~15
- 무시/거부 선택 → -5~10
- 해당 NPC의 조언을 따른 행동 → +10
- 해당 NPC의 조언을 무시한 행동 → -10

### 3.2 Spiritual Insight (영적 통찰)

성경 카드 수집과 해석자의 집 학습에 의해 누적되는 내부 수치.

| 트리거 | 변화량 |
|--------|--------|
| 성경 카드 수집 (일반) | +2 |
| 성경 카드 수집 (숨겨진 카드) | +5 |
| 해석자의 집 방 완료 | +3 |
| 올바른 분별 선택 (수다, 세상지혜씨 거부 등) | +5 |
| 잘못된 분별 선택 (유혹 수락) | -3 |

| 누적값 | 효과 |
|--------|------|
| 0–19 | 기본 서사만 경험 |
| 20–39 | NPC 본성에 대한 미묘한 힌트(내면 독백) |
| 40–59 | 환경 속 성경 구절이 빛나며 보임. 위험 지역 사전 경고 |
| 60+ | 크리스천의 내면 독백이 더 깊어짐. 최종 입성 연출 확장 |

### 3.3 Grace Counter (은혜 카운터)

잘못된 선택 후 전도자/구원의 개입이 발생할 때마다 +1. 표시되지 않으나 엔딩 연출에 반영.

| 값 | 엔딩 영향 |
|----|----------|
| 0–2 | "단단한 순례자" 칭호 가능성 |
| 3–5 | 기본 엔딩 |
| 6+ | 엔딩에서 은혜 강조 대사 추가 ("넘어질 때마다 일으키신 분") |

## 4. 스탯 상호작용 매트릭스

### 4.1 시너지 보너스

특정 스탯 조합이 높을 때 추가 효과 발생.

| 조합 | 조건 | 보너스 |
|------|------|--------|
| Faith + Courage | 둘 다 50+ | "담대한 신앙" — 유혹 저항 체크 시 Faith에 +10 보너스 적용 |
| Faith + Wisdom | 둘 다 50+ | "영적 분별" — 숨겨진 선택지 해금 임계값 -10 (더 쉽게 해금) |
| Courage + Wisdom | 둘 다 50+ | "지략" — QTE 실패 시 1회 재시도 기회 |
| 3스탯 모두 60+ | — | "성숙한 순례자" — NPC 추가 대화, 특별 성경 카드 1장 해금 |

### 4.2 트레이드오프

| 상황 | 설명 |
|------|------|
| Faith 높고 Courage 낮음 | "떨리는 믿음" — 올바른 선택은 하지만 도전 앞에서 주저하는 연출 |
| Courage 높고 Wisdom 낮음 | "무모한 용기" — 위험에 뛰어들지만 함정을 못 피하는 추가 이벤트 |
| Wisdom 높고 Faith 낮음 | "머리만의 지식" — 수다(Talkative)와 유사하다는 NPC 언급 |

### 4.3 Burden과 다른 스탯의 관계

| 조건 | 효과 |
|------|------|
| Burden ≥ 60 | Faith/Courage/Wisdom 획득량 ×0.8 (짐이 무거우면 성장이 더딤) |
| Burden ≥ 80 | Faith/Courage/Wisdom 획득량 ×0.6 |
| Burden = 0 (Ch6 이후) | Faith/Courage/Wisdom 획득량 ×1.2 (짐에서 자유로워진 성장 가속) |

## 7. 엔딩 분기 공식

### 7.1 최종 점수 계산

```
finalScore = (faith × 0.4) + (courage × 0.3) + (wisdom × 0.3)
graceBonus = min(graceCounter × 2, 10)
adjustedScore = finalScore + graceBonus
```

### 7.2 엔딩 분기

| 엔딩 | adjustedScore | Faith 최소 | 연출 |
|------|---------------|-----------|------|
| 영광의 입성 | 80+ | 80 | 나팔 소리, 금빛 파티클 폭발, 문이 활짝 열림, 환영 행렬, 면류관 수여 |

| 은혜의 구원 | 0–29 | 0 | 크리스천이 쓰러짐 → 빛나는 자들이 들어올림 → 은혜만으로 입성 → 가장 긴 엔딩 내레이션 |

## 9. 스탯 변화 연출 규칙

### 9.1 표시 방식

| 변화 크기 | 토스트 표시 | 추가 연출 |
|----------|-----------|----------|
| ±1~5 (미세) | "+3 믿음" 작은 텍스트 | 스탯 바 살짝 반짝임 |
| ±6~10 (보통) | "+8 용기!" 중간 텍스트 | 스탯 바 펄스 애니메이션 + 효과음 |
| ±11~15 (큰) | "+15 믿음!!" 큰 텍스트 + 파티클 | 스탯 바 강한 빛 + 화면 미세 흔들림 |
| ±16~20 (극대) | "+20 믿음!!!" 화면 중앙 대형 텍스트 | 전체 화면 빛 폭발 + BGM 크레센도 + 0.3초 히트스탑 |
| Burden → 0 | 특별 연출 (문서 4 참조) | 십자가 장면 전용 시퀀스 |

### 9.2 부정적 변화 연출

| 변화 | 연출 |
|------|------|
| 스탯 -1~-5 | 붉은 텍스트 "-3 용기", 스탯 바 붉게 깜빡 |
| 스탯 -6~-10 | 붉은 텍스트 + 화면 살짝 어두워짐 + 저음 효과음 |
| 스탯 -11 이상 | 붉은 비네트 + 화면 흔들림 + 경고음 |
| Burden 증가 | 짐 스프라이트 크기 증가 애니메이션 + 무거운 효과음 |
