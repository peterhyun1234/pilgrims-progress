| **Mount Sinai** | 시내산 | 율법. 율법으로는 짐을 벗을 수 없음 |

| **The Interpreter's House** | 해석자의 집 | 성령의 가르침을 받는 장소 |
| **The Cross** | 십자가 | 그리스도의 십자가 — 죄 사함의 장소 |
| **The Sepulchre** | 무덤 | 그리스도의 무덤 — 옛 사람의 죽음 |
| **Hill Difficulty** | 어려움의 언덕 | 신앙 생활의 어려움과 시련 |
| **The Arbour** | 정자 | 안일함의 유혹. 영적 졸음 |
| **The Palace Beautiful** | 아름다운 궁전 | 교회. 성도의 교제와 양육의 장소 |
| **Valley of Humiliation** | 겸손의 골짜기 | 겸손과 영적 전쟁의 장소 |
| **Valley of the Shadow of Death** | 사망의 음침한 골짜기 | 극한의 영적 암흑과 시험 (시 23:4) |
| **Vanity Fair** | 허영의 시장 | 세상의 욕망과 쾌락이 거래되는 곳 |
| **Plain of Ease** | 안일의 평지 | 영적 안일함에 빠지는 곳 |
| **Hill Lucre** | 이득의 언덕 | 물질적 탐욕의 유혹 (데마의 은광) |
| **Pillar of Salt** | 소금 기둥 | 롯의 아내. 뒤를 돌아본 자의 경고 |
| **River of God** | 하나님의 강 | 영적 휴식과 위로의 장소 |
| **By-path Meadow** | 샛길 초원 | 좁은 길에서 벗어나 쉬운 길을 택하는 유혹 |
| **Doubting Castle** | 의심의 성 | 의심과 절망에 갇힌 상태 |
| **Delectable Mountains** | 기쁨의 산 | 이마누엘의 땅. 영적 성숙과 조망의 장소 |
| **The Enchanted Ground** | 마법의 땅 | 영적 무감각과 졸음의 유혹 |
| **The Land of Beulah** | 블라의 나라 | 천국 직전의 평화로운 땅 (사 62:4) |
| **The River of Death** | 사망의 강 | 육체적 죽음 |
| **The Celestial City** | 천성 | 천국. 하나님의 도성 (히 12:22) |
| **Mount Zion** | 시온산 | 천상의 예루살렘 |
